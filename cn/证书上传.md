# 证书上传（SSL Certificate）

> 入口：CDNPro 控制台 → **Certificate Management** → **SSL Certificates** → **My Certificates**

把客户域名的 SSL/TLS 证书托管到网宿证书中心，并绑定到加速域名。

---

## 1. 物料准备

| 物料 | 说明 |
| --- | --- |
| 证书文件 | 支持 `.pem` / `.crt` / `.cer` / `.der` / `.pfx` / `.jks` |
| 私钥 | 与证书匹配的私钥；`.pfx` / `.jks` 含私钥时无需单独上传 |
| 私钥口令 | 仅当私钥加密 / `.pfx` / `.jks` 设了密码时需要 |
| 加速域名 | 证书 SAN 中包含的对外业务域名 |

---

## 2. 上传步骤

1. 进入 **My Certificates**，点击 **Add Certificate**。
2. 填写 **Certificate Name**。
3. 选择 **Key Algorithm**：
   - **国际标准（International Standard）**：RSA / ECC。
   - **国密 SM2**。
4. 上传 **Certificate** 与 **Private Key**；如有口令，填入 **Private Key Password**。
5. 在 **Associate Domain** 中勾选要绑定该证书的加速域名。
6. 提交。
