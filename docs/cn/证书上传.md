# 上传 SSL 证书

> 来源：Wangsu Documentation → Certificate Management → Tutorials → Upload an SSL Certificate（最后更新 2025-11-04）

## 概述

证书上传是指将 SSL 证书文件上传到证书管理平台，由平台分发并部署到全球边缘节点。

需要上传的文件包括：**证书文件**、**CA（证书颁发机构）文件**、**私钥文件**。

> **注意**
> - 上传证书仅完成证书文件的部署与分发，**系统不会自动**把已上传的证书关联到你的加速域名。要把证书绑定到加速域名，必须手动执行 **Associate Domain（关联域名）** 操作。
> - 如果上传方式不能满足你的安全或自动化需求，可以选择其他 SSL 证书创建方式。

---

## 操作步骤

### 1. 进入 My Certificates 页面

在 **Certificate Management Console** 左侧菜单点击 **SSL Certificates → My Certificates**，打开 **My Certificates** 页面。

![My Certificates 页面](../../assets/cert/cert-1.png)

### 2. 开始上传证书

点击证书列表左上角的 **Upload Certificate** 按钮打开上传表单，按以下要求填写信息。

#### 2.1 Upload Certificate（上传证书）

按要求填写证书文件信息。

- **Certificate Name（证书名称）**：用于标识和管理证书的自定义名称。
- **Notes（备注）**：可选，对证书的描述。
- **Key Algorithm（密钥算法）**：选择证书所使用的加密算法类型。所需上传的证书文件取决于你的选择。
  - **Certificate – International Standard（国际标准证书）**：如果证书使用国际通用算法（如 RSA 或 ECC），选此项。需上传**证书文件**、**CA 文件**、**私钥文件**。
  - **SM2 Certificate – Chinese National Standard（国密 SM2 证书）**：如果证书使用国密 SM2 算法，选此项。需同时上传 **SM2 签名证书**（签名证书文件、CA 文件、签名私钥文件）和 **SM2 加密证书**（加密证书文件、CA 文件、加密私钥文件）。
- **Upload Certificate by（上传方式）**：
  - **Import from Certificate File（从证书文件导入）**：直接上传本地证书文件，选此项后点击 **Upload**。支持格式：`.pem`、`.key`、`.crt`、`.cer`、`.der`、`.pfx`、`.jks`。每个证书文件不能超过 44 KB。
  - **Paste Certificate（粘贴证书）**：手动录入证书内容，选此项后在 **Certificate Content（证书内容）**、**Private Key Content（私钥内容）**、**CA Content（CA 内容）** 文本框里粘贴。**仅支持 PEM 编码格式。**

#### 2.2 Certificate Parsing（证书解析）

系统会自动解析并校验证书，并显示以下信息：

| 字段 | 说明 |
| --- | --- |
| **Authorized Domain(s)** | 证书授权的域名，包含 Common Name (CN) 和 Subject Alternative Names (SAN)。 |
| **Serial Number** | 证书的唯一序列号。 |
| **Issuer** | 证书签发者信息，使用 CA 的 Distinguished Name (DN) 表示，可能包含 Common Name (CN)、Organization (O)、Country Code (C)、Location/City (L)。 |
| **Issued To** | 证书颁发对象的信息，通常包含 Common Name (CN) 和 Subject Alternative Name (SAN)。 |
| **Issue Date** | 证书的生效日期。 |
| **Expiration Date** | 证书的到期日期。 |
| **Encryption Algorithm** | 证书使用的签名算法。 |
| **Key Usage** | 证书密钥的用途，如签名（Signature）、加密（Encryption）或两者皆用。 |

![上传表单与证书解析结果](../../assets/cert/cert-2.png)

### 3. Submit（提交）

确认证书上传表单无误后，点击页面底部 **Submit** 按钮。

### 4. View the Result（查看结果）

提交成功后，系统会自动跳转到 **My Certificates** 列表页，可在列表中找到已上传的证书。如果 **Associated Domains** 列为空，表示当前没有任何域名使用该证书，需要执行 **Associate Domain（关联域名）** 操作来给域名启用此证书。
